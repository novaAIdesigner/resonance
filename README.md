# Resonance

Resonance 是一个基于 Azure Speech 应用场景的专属产品营销网站（静态模块 + 页面组件，无 CMS）。

## 技术栈

- Vite
- React + TypeScript
- Tailwind CSS

## 页面能力

- 首页最新发布轮播导向产品单页
- 产品价值导向（高性能/低成本/全语言）
- 场景化 Demo（通过服务端 API 路由占位）
- 客户案例与证言轮播
- 2C 登录与充值入口、2B Azure 入口
- Developer 与 API 文档入口
- 安全、合规、RAI 说明

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
